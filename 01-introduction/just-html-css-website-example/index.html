How does HTML work? ü§î

An HTML (Hypertext Markup Language) document outlines the structure of a web page using tags.

Tags in an HTML document tell the browser how to display our text, images, and links. For example, there are several different tags that represent text headings, similar to the different headings you might use in a Word document. They are called h1, h2, h3, h4, etc. There is also a tag for paragraph text, like the text you are reading right now. This tag is simply called p.

Tag syntax helps the browser to understand what structure you'd like the webpage to have - where headings start and end, and where paragraphs begin and finish. Like this: 

<h1> Example Heading </h1>
<p> This is an example of paragraph text </p>

You can see that the starting tag looks like <p> whereas the closing tag looks like </p>. The / represents the closing tag.

To add images, we use the <img> tag. Unlike paragraphs and headings, we can close within the same tag for images. For example, the code for the image below is: 

<img class="main-page-image" src="https://picsum.photos/id/16/2000/1000" alt="My test image"/>

This highlights the slight differences in syntax between tags. It's okay if you don't remember these off the top of your head. Even developers with a lot of experience find themselves Googling things about HTML tags.

Using just HTML alone, you can create a readable webpage. However, CSS can help us add styling. This can further enhance readability and generally make the page look a lot more appealing!

To learn more about HTML, have a look at Mozilla Developer Docs here.

How does CSS work? ü§î

CSS (Cascading Style Sheets) allows us to describe how we want our page to look using rulesets.

Rulesets, often referred to as just rules, allow us to describe what we want specific elements of our HTML document to look like. They look like this: 

p { colour: red };

In this example, we are saying we want all paragraphs to have red text. p is the CSS 'selector' and maps directly to our HTML tag, color is the property we are defining, red is value we want to assign the property.

To learn more about CSS, have a look at Mozilla Developer Docs here.
